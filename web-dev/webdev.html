Travel and Explore Now

A new travel website that integrates ways to improve their effectiveness. This includes advanced search
filters, natural language processing for searching and natural language enabled virtual assistant. The aim
of this project is to present and implement an online platform with advanced services for users looking
for holidays. A secondary aim of this study is to provide an insight into the main components of online
travel websites and inform their future design and practice.
Technologies Used: MVC PHP Laravel 3 framework, HTML, CSS/Bootstrap. Javascript/jQuery, JSON for
server-side and client-side data parsing, AJAX, XML. DBMS MySQL, Self developed APIs and
Paypal Payments.
